# Quickstart: Advanced To-Do App

This guide will help you set up and run the Advanced To-Do App locally.

## Prerequisites

*   Node.js (v20 or higher)
*   npm or Yarn (preferably npm, included with Node.js)
*   Git

## 1. Clone the Repository

```bash
git clone https://github.com/ShehrozHanif/todo_gemini.git
cd todo_gemini
git checkout 1-todo-app-advanced
```

## 2. Install Dependencies

```bash
npm install
# or yarn install
```

## 3. Run the Development Server

```bash
npm run dev
# or yarn dev
```

This will start the development server on `http://localhost:3000`.

## 4. Build for Production

```bash
npm run build
# or yarn build
```

This command builds the application for production.

## 5. Start Production Server

```bash
npm start
# or yarn start
```

This will start the production server after building.

## 6. Testing

To run the tests:

```bash
npm test
# or yarn test
```
